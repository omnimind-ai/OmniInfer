# OmniInfer

This repository is organized as a lightweight workspace for platform integration,
framework submodules, tests, and project documentation.

## Structure

- `platform/`: platform-specific integration entry points
- `framwork/`: upstream and OmniInfer framework submodules
- `tests/`: repository-level test guidance and validation notes
- `docs/`: project documentation and architecture notes

- `framwork/llama.cpp`
- `framwork/mlx`
